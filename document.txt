# 🛒 Multi-Vendor QuickCommerce API

A production-ready **Multi-Vendor E-Commerce REST API** built with **Spring Boot 3**, following a layered architecture and RESTful design principles.
The platform enables multiple vendors to manage products, inventory, orders, and customers while providing secure authentication and scalable backend services.

---

## ✨ Key Features

* 🔐 JWT Authentication & Role-Based Authorization
* 🏪 Multi-Vendor Marketplace
* 📦 Product & Inventory Management
* 🛍️ Shopping Cart & Wishlist
* 📂 Category Management
* 📍 Customer Address Management
* 📑 Order Processing
* ⭐ Product Reviews & Ratings
* 📊 Vendor Dashboard APIs
* 🚚 Delivery Tracking Support
* ⚠️ Global Exception Handling
* 🔄 DTO Mapping & Validation
* 🌐 RESTful API Architecture

## 🚀 Getting Started

### Configure the Application

Create:

src/main/resources/application.properties


spring.datasource.url=jdbc:mysql://localhost:3306/quickcommerce
spring.datasource.username=root
spring.datasource.password=your_password

spring.jpa.hibernate.ddl-auto=update

jwt.secret=your_secret_key
jwt.expiration=86400000

> **Security Notice:** Never commit `application.properties`. Use `application.properties.example` as a template and add the actual configuration file to `.gitignore`.

### Build

mvn clean install

### Run

mvn spring-boot:run

or start `QuickecommerceApplication.java` directly from your IDE.

## 🔐 Authentication

The API uses **JWT Bearer Authentication**.

Authentication Flow:

Register
     │
     ▼
 Login
     │
     ▼
 Receive JWT Token
     │
     ▼
Authorization: Bearer <JWT_TOKEN>

## 📌 Sample API Endpoints

| Method | Endpoint             | Description         |
| ------ | -------------------- | ------------------- |
| POST   | `/api/auth/register` | Register a new user |
| POST   | `/api/auth/login`    | Authenticate user   |
| GET    | `/api/products`      | Retrieve products   |
| POST   | `/api/products`      | Create a product    |
| GET    | `/api/orders`        | Retrieve orders     |
| POST   | `/api/orders`        | Place an order      |
